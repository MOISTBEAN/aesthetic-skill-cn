# Premium AI SaaS Preview

## Representative screen

**Scenario:** A procurement analyst reviews `华东仓配 2026 年续约合同`.

The desktop preview uses a dark three-zone workstation:

- left: project files and analysis history
- center: `原合同` and `AI 修改建议` in a synchronized comparison
- right: source clauses, comments, run details, and reviewer status

The header reads `发现 4 处条款差异 · 2 处待法务确认`. One warning states `第 8 页扫描质量较低`; it does not pretend the run fully succeeded. The primary action is `提交法务复核`.

## Product preview wall

Show three connected surfaces at readable scale:

1. contract comparison workstation
2. cited clause inspector
3. analysis-history and approval state

Use neutral charcoal surfaces, crisp borders, restrained blue actions, and violet AI-provenance details. Avoid an ornamental hero, robots, glass cards, or unreadable miniature dashboards.
