# aesthetic-skill-cn

An AI-readable design skill for Cursor, Claude Code, Codex, and other coding agents. It helps agents produce credible Chinese product interfaces instead of generic landing-page theater.

> Stop generating fake SaaS UI. Start building real Chinese product interfaces.

## Start

Point your coding agent to `SKILL.md`, describe the real product context, and choose a preset from `design-md/`. See `docs/installation.md` for setup notes.

This repository contains design rules, review criteria, portable `DESIGN.md` presets, and small before/after references—not a SaaS application.
