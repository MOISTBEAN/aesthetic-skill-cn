# Non-AI Style Scorecard

Score each category from 1 to 10 using visible evidence. Do not score intent.

| Category | 1-3 | 4-5 | 6-7 | 8-10 |
|---|---|---|---|---|
| Product Specificity | Renameable template | A few domain labels | Several real modules | Product cannot survive the Product Swap Test |
| Workflow Reality | Display only | Isolated actions | Partial workflow | Check, compare, decide, act, and revisit are connected |
| Copy Naturalness | Translated slogans | Mixed abstract copy | Mostly concrete | Native, concise, domain-correct actions and states |
| Negative State Presence | All positive | Decorative warning | One actionable exception | Multiple relevant states with recovery or review actions |
| Decision Support | No decision | Metrics imply action | Decision context exists | Evidence, tradeoff, status, and next action are explicit |
| Visual Non-Genericness | AI template tells dominate | Generic shell | Product structure leads | Visual system follows unique workflow and density |

## Passing rule

- Average score must be `>= 8.0`.
- Any category below `6` fails the output.
- Cite one visible fact for every score.
- A high average cannot compensate for a failed category.

## Template

```text
Product Specificity: 9/10 - ...
Workflow Reality: 8/10 - ...
Copy Naturalness: 8/10 - ...
Negative State Presence: 9/10 - ...
Decision Support: 8/10 - ...
Visual Non-Genericness: 8/10 - ...
Average: 8.3/10
Result: PASS
```
